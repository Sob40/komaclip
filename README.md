# KomaClip

Create social promo clips from comics and webtoons.

KomaClip is being rebuilt as a production SaaS from the Panel2Reels MVP. Rails owns the product, billing, users, credits, projects, jobs, and admin. Pixi is the single visual renderer for previews and exports.

## Stack

- Rails 8.1
- PostgreSQL
- Sidekiq and Redis / Valkey
- Pixi renderer
- Polar billing
- Cloudflare R2 or S3 object storage
- Tailwind CSS and esbuild

## Local Setup

```bash
bundle install
yarn install
bin/rails db:prepare
bin/dev
```

Pixi is the visual source of truth. The browser previews quickly, but paid exports are generated by trusted workers.
